pycon2016

Having a difficult time deciding which talks to go on PyCon 2016 Ireland?

Download the iCalendar file pycon2016.ics and import it to your favorite Calendar application (Google Calendar, MS Outlook, Mozilla Thunderbird, etc.)

Download .ics

If you want the most up-to-date version then you should create it by yourself, simply download the code and run the main.py script, that'll create a pycon2016.ics file that you can import to your Calendar application.

Enjoy!
